# Livepeer Explorer

## Getting started

1. Rename .env.example to .env
2. Install it and run:

```bash
yarn
yarn dev
```

## Developing on Arbitrum Rinkeby

Set `NEXT_PUBLIC_NETWORK` to `ARBITRUM_RINKEBY` inside `.env`
